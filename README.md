# Matching Aspiration Pro-AI

吸引カテーテル参照 × コアキシャル音声ガイドツール（脳血管内治療補助用）

## 概要

脳血管内治療において使用する吸引カテーテルの選択をサポートし、  
AI音声ガイド機能を搭載したWebアプリケーションです。  
PWA（Progressive Web App）対応のため、ホーム画面に追加してネイティブアプリのように利用できます。

## ファイル構成

```
├── index.html          # メインアプリケーション（v2.7）
├── manifest.json       # PWAマニフェスト
├── netlify.toml        # Netlify 設定ファイル
├── 404.html            # 404エラーページ
├── README.md           # 本ファイル
└── icons/
    ├── apple-touch-icon.png   # iOS ホーム画面アイコン (180×180)
    ├── icon-192.png           # Android ホーム画面アイコン (192×192)
    ├── icon-512.png           # Android スプラッシュ用アイコン (512×512)
    ├── icon-master.png        # マスターアイコン (1024×1024)
    └── icon.svg               # SVGアイコン
```

## デプロイ

GitHub リポジトリと連携し、以下のサービスへ自動デプロイしています。

- **Netlify**
- **Cloudflare Pages**

`main` ブランチへのプッシュで自動的にデプロイが実行されます。

## PWA（ホーム画面追加）

iOS Safari・Android Chrome から「ホーム画面に追加」することで、  
アプリとして起動できます。アイコンには本ツール専用デザイン（Pro-AI版）が設定されています。

> **Note:** キャッシュが残っている場合は、一度ホーム画面から削除して再追加すると新しいアイコンが反映されます。

## パスワード変更方法

認証パスワードはSHA-256ハッシュで管理されています。  
`index.html` 内の `_PH` 定数を新しいパスワードのSHA-256ハッシュ値に更新してください。

```bash
# ハッシュ生成例（ターミナル）
echo -n "新しいパスワード" | shasum -a 256
```
